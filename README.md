# Chasse au Trésor — Tremblant 🏔️🎂

Birthday treasure hunt for **Valérie** at Mont-Tremblant — family-friendly with **Émile**, from Holiday Inn breakfast to **A Mano** dinner.

## Deploy on Vercel (share the link)

**Fastest way** — import the repo (takes ~1 minute):

👉 **[Deploy on Vercel](https://vercel.com/new/clone?repository-url=https://github.com/philthebig/valBirthday&project-name=val-birthday-tremblant)**

Click Deploy, then share the `*.vercel.app` URL with Valérie.

## Run locally

```bash
npm install
npm run dev
```
